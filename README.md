## FUNCTION
<br>

<br>

1. **Create**: 데이터 생성
   > 지출 항목이나 비용이 비어있을 경우: 에러 Alert! 
   > 

2. **Read**: 데이터 읽기 -> Load: 보여주기까지

3. **Update**: 데이터 수정

4. **Delete**: 데이터 삭제
